Operating Model / 002

Your Operating Model

Centralised labour control with one key principle: Reallocation First

01

Central Labour Control

All labour allocation flows through a single point of control, ensuring consistency, compliance, and cost management.

02

Reallocation First

Before any external recruitment, the system searches internal workforce for available, qualified operatives.

03

Skills & Rate Governance

Every allocation checks skill requirements against operative qualifications and enforces rate matrices.

04

Compliance by Default

No operative works without valid documents. The system blocks non-compliant allocations automatically.

Sophie AI Agent / 005

Sophie AI Agent

Your AI-powered recruitment assistant that qualifies candidates 24/7.

What Sophie Does

Responds to applicant enquiries via WhatsApp
Collects skills, experience, and certifications
Validates CSCS cards and expiry dates
Checks availability and location preferences
Qualifies candidates before reaching Labour Manager
Schedules document uploads and follow-ups

Qualifying Logic

✓ Qualified

Valid CSCS + Right to Work + Required Skills → Passed to Labour Manager

✗ Not Qualified

Missing documents or skills → Stored for future roles, requested to complete

~£0.30 per applicant vs £500+ agency fee per placement

Red Card System

Automatic Enforcement

NCR triggers automatic review. Multiple NCRs = blocked status. "Would Re-employ" flag from supervisors. Blocked operatives excluded from all searches.

Critical NCR = Instant Block

Drugs/alcohol or serious safety breach immediately blocks operative.
